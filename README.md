Конвейер с защитой. 

Как работает?
1)Нажимаем кнопку «Пуск»
2)Таймер отсчитывает 2 секунды (на экране бегут цифры 0.0 → 2.0)
3)Через 2 секунды включается двигатель конвейера
4)Двигатель работает, пока не нажмут «Стоп» или не сработает концевик
5)Чтобы запустить снова — нужно опять нажать «Пуск»

Защита:
- Стоп — мгновенно выключает двигатель
- Концевик — выключает двигатель при срабатывании (имитация опасной зоны)
